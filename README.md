# power-bi-dashboard
Employees Analytics
Power BI Employee Analytics Dashboard
Project Overview
This project presents an Employee Analytics Dashboard created using Power BI.
The dashboard helps analyze employee data such as department distribution, salary trends, employee performance, and overall workforce insights.

Tools Used
- Power BI
- Microsoft Excel
- Data Visualization

Dataset
The dataset contains employee information such as:
- Employee ID
- Name
- Department
- position
- Salary
- Experience
-city

Dashboard Features
- Total Employees KPI
- Department-wise employee distribution
- Salary analysis
- Interactive filters and visuals

Purpose
The purpose of this dashboard is to help organizations understand employee data and make better HR decisions.
